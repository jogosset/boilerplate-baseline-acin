# Carousel Block

## Overview

The Carousel Block provides a responsive carousel-3/slider component for displaying multiple items in a rotating slideshow format.

## Features

- Auto-rotation with configurable interval
- Navigation controls (previous/next buttons)
- Pagination indicators
- Touch/swipe gesture support
- Keyboard navigation
- Pause on hover
- Responsive design
- Accessibility support

## Usage

The carousel-3 block automatically initializes when added to a page. Structure your content in a table format:

| carousel-3 |
|----------|
| Slide 1  |
| Slide 2  |
| Slide 3  |
